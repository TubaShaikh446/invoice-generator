# 🧾 Invoice Generator Web Application

## 📌 Tech Stack
- Spring Boot
- Java 17
- MySQL
- HTML, CSS, JavaScript
- Thymeleaf

---

## 🚀 Features
- Create Invoice
- Dynamic Item Rows (Add/Remove)
- Auto Calculations (GST, Total, Grand Total)
- Save Invoice to Database
- View Invoice List
- Print Invoice

---

## 🗄 Database Tables
- invoices
- invoice_items
- company_details

---

## ⚙️ How to Run
1. Clone the repository
2. Import in Spring Tool Suite / IntelliJ
3. Setup MySQL database
4. Update `application.properties`
5. Run Spring Boot Application
6. Open: `http://localhost:8080`
